## 项目背景
```
该项目是一个偏向于金融行业CRM后台管理系统，主要是为了将自己学习的新技术运用到项目中，加油😬！
```

## 项目描述
```
Vue+Vuex+Axios+IView UI 前后分离项目，Vue-Cli 3.0 脚手架，后端接口是.Net Core WebApi。 
```

## 安装依赖项
```
npm install
```

### 开发环境运行此命令行
```
npm run serve
```

### 生产环境运行此命令行
```
npm run build
```

### EsLint根据ES Ruler自动修复
```
npm run lint
```
