# vuedemo

## 项目描述
```
Vue+Vuex+Axios+IView UI 前后分离项目，Vue Cli 3.0脚手架，后端接口是WebApi。 
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
